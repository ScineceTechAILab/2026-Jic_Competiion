# 硬件配置与选型说明

## 1. 核心计算单元
- **型号**: 地瓜机器人 RDK X5 8GB
- **系统版本**: Ubuntu 22.04
- **供电要求**: Type-C 5V/3A (或具体电压电流)

## 2. 移动底盘
- **类型**:两轮差速轮（后轮）+两轮全向轮（前轮）
- **电机驱动**: 2 个 TB6612FNG 电机驱动模块
- **编码器**: 2 个 QEI 编码器
- **轮径**: 12.5 cm
- **轮间距**: 25 cm

## 3. 机械臂系统

- **自由度**: 2 DOF
- **舵机型号**: 2 个 SG90 舵机
- **末端执行器**: 夹爪（注射器气动夹持）
- **控制驱动板**：树莓派扩展板电机驱动板
- **控制接口**: IIC



## 4. 传感器配置
### 视觉
- **摄像头型号**: Astra Pro SM
- **安装位置**: 车顶部
- **分辨率/帧率**: 1280x720 @ 30fps

> 提示：详情见 https://ocn6ppgv7e2y.feishu.cn/wiki/ZlrswanpPiwoHEkxLWKc5cVInMd;https://ocn6ppgv7e2y.feishu.cn/wiki/MLatwtcaTiSHSmkN3OcclLKGnIf 及其子文档

### 激光雷达 (LiDAR)
- **型号**: RPLIDAR S1
- **串口/USB**: 串口
- **波特率**: 115200

### IMU
- **型号**: BNO055（陀螺仪+加速度计+磁力计）
- **通讯协议**: I2C 
- **安装方向**: 车后方中轴线处

## 5. 接口与连线定义 (Pinout)
| 模块 | RDK 接口 | 引脚号 | 备注 |
| :--- | :--- | :--- | :--- |
| 底盘 MCU | UART | | |
| 激光雷达 | USB | | |
| 摄像头 | MIPI / USB | | |

## 6. 电源管理
- **电池规格**: 24V 8S 3700mAh 电池组
- **稳压模块**: 2 个 LM2596 稳压模块
- **充电接口**: Type-C 5V/3A 充电接口
